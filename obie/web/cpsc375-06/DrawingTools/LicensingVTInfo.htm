Here ia copy of the old FAQ. Most of the information in this document concerning licencing is no longer relevant. Please see below for the latest FAQ concerning licencing issues.

FAQ

  1. Q: Will this download ever expire?

    A: When you download Visual Thought you are automatically given a 13-year license to use the product.

  2. Q: Do I need a server code?

    A: No, the license key used to unlock your copy of Visual Thought is server code (machine code) independent.

  3. Q: Is the product still supported?

    A: No. Visual Thought is no longer supported. This applies to phone, email, fax and upgrade support.

  4. Q: I have questions on how to use Visual Thought...

    A: The Visual Thought User's Guide is opened by clicking the Help/User's Guide button. Hardcopies of the User's Guide are NOT available (unless you want to print it yourself.) For more on using the program, please see the Visual Thought Guided Tour. The Guided Tour is opened by clicking the Help/Overview button from the Visual Thought standard toolbar.
  5. Q: Will there be future versions of this product?

    A: No. Visual Thought will only exist as it does today. Future versions, ports and functionality will no longer be made available.
  6. Q: How much does it cost?

    A: Visual Thought is Freeware. This means that you may download and use Visual Thought to your heart's content. All features that previously required a license key have been unlocked for this version of the product.

  7. Q: Can I get a hard copy of the software? (or: Can I get this program on disc?)

    A: No. Hardcopies of the software or user manual are NOT available.

  8. Q: How do I download it?

    At CERN the software is available in AFS and on DFS (see the relevant pages). Otherwise to begin the download process via ftp protocol, click here.

  9. Q: How do I install Visual Thought?

    A: Double click the Visual4.exe once you've downloaded it and follow the on-screen instructions. You'll also need the license key, which is available on the downloads page.

  10. Q: Can I redistribute Visual Thought?
    A: Yes, you may redistribute the program as long you do not modify any of the source code.
